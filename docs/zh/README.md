features:
- title: 介绍
  details: Webterminal是一款为实现自动运维化持续集成及集中化管理IT设备而创建的统一集中化管理平台。
- title: 环境
  details: 前后端分离、前端基于Vue quasar 后端Python django、qt、c++、golang进行开发
- title: 权限
  details: 用户权限基于django内置自定义权限，支持开启用户MFA认证等
- title: RDP
  details: 支持网页端管理(文件上传、下载、远程连接、实时监控、下线控制、事后审计、用户行为录像),支持本地mstsc原生调用，实现NLA功能(商业版本提供)
- title: VNC
  details:  支持网页端管理(文件上传、下载、远程连接、实时监控、下线控制、事后审计、用户行为录像),支持本地mstsc原生调用(商业版本提供)
- title: SSH
  details: 支持网页端管理(文件上传、下载、远程连接、实时监控、下线控制、命令审计、事后审计、用户行为录像),支持不限于putty、xshell、securecrt等工具远程直接调用连接
- title: SSH批量执行
  details: 支持批量Linux服务器执行命令、脚本、实时监控、实时控制及任务
- title: Sftp
  details: 支持网页端管理(文件上传、下载、更新、删除、文件夹上传、文件夹打包下载、在线解包),支持不限于FileZilla等工具远程直接调用连接
- title: Telnet
  details: 支持网页端管理(远程连接、实时监控、下线控制、事后审计、用户行为录像)
